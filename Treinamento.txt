GIT HUB 

Esse material é para executar os comandos mais utilizados no git

Segue a baixo os comandos que são necessário para começar executar o git .

echo "# TreinamentoGitHub" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:bgfenelon/TreinamentoGitHub.git
git push -u origin main

teste na develop
